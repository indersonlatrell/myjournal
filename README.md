# MyJournal (Vite + React)

Simple Vite React app to preview the `Check` trading dashboard component.

Prerequisites
- Node.js (use `nvm install --lts` if you don't have Node)

Run locally

```bash
# from project root
nvm install --lts    # if needed
npm install
npm run dev
```

Open http://localhost:5173/ in your browser.

Files of interest
- `src/Check.jsx` — adapted dashboard component (from `check.react.js`)
- `src/components/ui/` — minimal UI stubs to run the component
